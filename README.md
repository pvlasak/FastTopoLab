<h3> This project is about to develop VueJS application with backend running in Node.js to present portfolio of startup company focused on digital simulations, topology optimization and additive manufacturing.</h3> 

<h2> Steps to start the application: </h2>
<p> Prerequisities: npm and node must be installed. </p> 
<ul> 
    <li> npm install - installs all dependencies in package.json available in node_modules folder </li>
    <li> npm run build - builds the minified version of VueJS frontend and creates dist directory </li>
    <li> npm run start - starts the application and backend server listens on port nr. 3000 </li>
</ul>
